# 🔘 DECK BUILDER BUTTON ASSETS

Upload your custom deck builder button images here!

## 🎮 Required Button Assets:

### **Deck Management Buttons:**
- `create-deck-btn.png` - "➕ New Deck" button
- `deck-selector-btn.png` - "📋 Decks" dropdown button  
- `save-deck-btn.png` - Save deck button
- `delete-deck-btn.png` - Delete deck button
- `set-active-deck-btn.png` - "Set as Active" button

### **Card Action Buttons:**
- `add-to-deck-btn.png` - "+" button for adding cards to deck
- `remove-from-deck-btn.png` - "-" button for removing cards from deck
- `card-details-btn.png` - Button to view card details

### **📏 Recommended Specifications:**
- **Format:** PNG with transparency
- **Size:** 128x64 pixels for rectangular buttons
- **Size:** 48x48 pixels for square icon buttons (+/-)
- **Style:** Match your game's button style
- **States:** Consider normal, hover, pressed variations

### **🎯 Button Layout:**
```
Top Management Bar:
[create-deck-btn] [deck-selector-btn] [save-deck-btn] [delete-deck-btn]

Collection Cards:                Deck Cards:
┌─────────────────┐             ┌─────────────────┐
│ Fire Potato     │             │ Fire Potato     │
│ [add-to-deck]   │             │ [remove-from]   │
└─────────────────┘             └─────────────────┘
```

### **🎨 Design Tips:**
- **Create Deck:** Green/positive color
- **Delete Deck:** Red/warning color  
- **Save Deck:** Blue/confirmation color
- **Set Active:** Gold/special color
- **Add Card:** Green plus icon
- **Remove Card:** Red minus icon
