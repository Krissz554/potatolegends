# 🔤 CUSTOM FONTS FOLDER

## 📁 **Upload Your Custom Fonts Here:**

### **🎮 Font Types:**
- `fantasy-title-font.ttf` - Fantasy font for game titles
- `clean-body-font.ttf` - Clean font for body text and UI
- `pixel-font.ttf` - Pixel font for retro elements (optional)

### **📝 Font Usage:**
- **Fantasy Title Font** - Game titles, screen headers, card names
- **Clean Body Font** - Button text, descriptions, stats
- **Pixel Font** - Special elements, easter eggs

## 🎨 **Font Requirements:**
- **TTF or OTF** format
- **Unicode support** for special characters
- **Good readability** at small sizes (mobile)
- **Consistent style** with your game theme

## 🔧 **Import Settings:**
After adding fonts:
1. Select the font in Unity
2. Inspector → Font Settings
3. Set appropriate font size range
4. Click "Apply"

## 📱 **Mobile Considerations:**
- **High contrast** for mobile screens
- **Good readability** at various sizes
- **Not too decorative** for body text
- **Fantasy feel** for titles without sacrificing readability

## ✅ **TextMeshPro Integration:**
Unity will automatically create TextMeshPro font assets from your TTF files for better mobile performance and effects.

## 🎯 **Priority:**
Custom fonts are optional - the current system uses professional default fonts that work well. Add custom fonts only if you want specific branding.
