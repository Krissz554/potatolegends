# ⚡ CARD STAT & UI ICON ASSETS

Upload your custom card stat and UI icons here!

## 📊 Required Card Stat Icons:

### **Card Statistics Icons:**
- `mana-crystal-icon.png` - Mana cost icon (blue crystal/gem)
- `attack-icon.png` - Attack stat icon (sword/weapon)
- `health-icon.png` - Health stat icon (heart/shield)

### **Deck Builder UI Icons:**
- `deck-count-icon.png` - Card count icon for "15/30" display
- `active-deck-crown.png` - Crown icon for active deck indicator
- `deck-validation-check.png` - Checkmark for valid decks
- `deck-validation-error.png` - X mark for invalid decks

### **📏 Recommended Specifications:**
- **Format:** PNG with transparency
- **Size:** 32x32 pixels for stat icons
- **Size:** 24x24 pixels for small UI icons
- **Style:** Clear, readable at small sizes
- **Colors:** Match your game's color scheme

### **🎯 How They're Used:**
```
Card Display:
┌─────────────────────┐
│ Fire Potato         │
│ [mana] 3  [atk] 4   │  ← Your custom icons here
│ [health] 5          │
└─────────────────────┘

Deck Management:
👑 Active: MyDeck [deck-count] 15/30 ✓
   ^crown    ^count-icon    ^validation
```

### **🎨 Icon Design Tips:**
- **Mana:** Blue crystal, gem, or magical symbol
- **Attack:** Red sword, claw, or weapon
- **Health:** Green heart, shield, or cross
- **Crown:** Gold crown for active deck status
- **Count:** Small number or card stack icon

### **🌈 Color Coding:**
- **Mana:** Blue (#4169E1)
- **Attack:** Red (#DC143C)  
- **Health:** Green (#32CD32)
- **Active:** Gold (#FFD700)
- **Valid:** Green (#00FF00)
- **Invalid:** Red (#FF0000)
