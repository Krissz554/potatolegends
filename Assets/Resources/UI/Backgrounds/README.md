# 🖼️ CUSTOM BACKGROUNDS FOLDER

## 📁 **Upload Your Custom Backgrounds Here:**

### **🎨 Screen Backgrounds:**
- `main-menu-bg.png` - **Your custom main screen background** (PRIORITY!)
- `auth-bg.png` - Auth/login screen background (optional)
- `collection-bg.png` - Collection screen background (optional)
- `deck-builder-bg.png` - Deck builder screen background (optional)
- `hero-hall-bg.png` - Hero hall screen background (optional)
- `battle-bg.png` - Battle arena background (optional)

### **🎭 Atmospheric Backgrounds:**
- `gradient-overlay.png` - Gradient overlay for depth effects
- `particle-bg.png` - Background for particle effects
- `texture-overlay.png` - Subtle texture overlay

## 📐 **Recommended Sizes:**
- **Mobile Backgrounds:** 1080x1920px (16:9 portrait ratio)
- **Tablet Support:** 1440x2560px (for larger screens)
- **Minimum:** 720x1280px (will scale up)

## 🎨 **Format & Style:**
- **PNG or JPG** format
- **High resolution** for crisp mobile display
- **Fantasy/Elemental theme** matching your game
- **Proper contrast** for UI elements on top
- **Consistent art style** across all backgrounds

## 🎯 **Background Usage:**
- **main-menu-bg.png** - Used as fullscreen background for main menu
- **auth-bg.png** - Used as fullscreen background for login screen
- **Other backgrounds** - Used for their respective screens

## 🔧 **Import Settings:**
After adding your backgrounds:
1. Select the background in Unity
2. Inspector → Texture Type → "Sprite (2D and UI)"
3. Inspector → Sprite Mode → "Single"
4. Click "Apply"

## ✅ **Automatic Integration:**
The ProductionUIManager will automatically detect and use your custom backgrounds when you add them to this folder!

## 🎮 **Priority Asset:**
**`main-menu-bg.png`** is the most important - this will transform your main menu to look exactly like your vision!
