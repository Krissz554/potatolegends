## 📐 **Recommended Sizes:**
- **Card Frames:** 300x400px (standard card ratio)
- **Hero Cards:** 350x450px (slightly larger)
- **Stat Icons:** 64x64px (mana, attack, health)
- **Overlay Effects:** 300x400px (same as card frames)

## 🎨 **Design Guidelines:**
- **Transparent backgrounds** with decorative frames
- **9-slice ready** borders for different card sizes
- **Fantasy theme** consistent with your game
- **Clear visual hierarchy** (frame, content, stats)
- **Good contrast** for text and numbers
