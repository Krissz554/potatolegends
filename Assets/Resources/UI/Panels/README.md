# 🎭 CUSTOM PANELS FOLDER

## 📁 **Upload Your Custom Panel Frames Here:**

### **🎨 Panel Types:**
- `auth-panel-frame.png` - Frame for login/register panel
- `header-panel.png` - Frame for screen headers
- `navigation-card.png` - Frame for navigation cards
- `content-panel.png` - Frame for content areas
- `filter-panel.png` - Frame for filter bars

### **🃏 Card-Related Panels:**
- `card-frame.png` - Frame for individual cards
- `hero-card-frame.png` - Special frame for hero cards
- `deck-slot-frame.png` - Frame for deck building slots

### **🔧 Utility Panels:**
- `utility-bar.png` - Frame for bottom utility bar
- `currency-panel.png` - Frame for gold/gems display
- `stats-panel.png` - Frame for player stats

## 📐 **Recommended Sizes:**
- **Large Panels:** 600x400px (auth panel, content areas)
- **Header Panels:** 800x100px (screen headers)
- **Navigation Cards:** 250x150px (menu navigation)
- **Small Panels:** 200x100px (utility elements)

## 🎨 **Design Guidelines:**
- **9-slice ready** borders for scaling
- **Fantasy/game theme** styling
- **Transparent backgrounds** with decorative borders
- **Consistent visual style** across all panels
- **Good contrast** for content readability

## 🔧 **9-Slice Border Setup:**
Your panel frames should have:
- **Decorative corners** (don't stretch)
- **Stretchable sides** (can repeat/stretch)
- **Transparent center** (for content)

## ✅ **Automatic Integration:**
The ProductionUIManager will automatically detect and use your custom panel frames when you add them to this folder!

## 🎯 **Priority:**
Panel frames are optional but add significant visual polish. The current system uses beautiful solid panels with gradients and shadows.
